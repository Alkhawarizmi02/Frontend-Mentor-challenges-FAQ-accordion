FAQs

What is Frontend Mentor, and how will it help me?
Frontend Mentor is a platform that provides real-world front-end coding challenges. These challenges help you practice and improve your coding skills by building projects that replicate real-world scenarios. You'll work with HTML, CSS, and JavaScript to build projects from designs provided by Frontend Mentor.
How do I start a challenge on Frontend Mentor?
To start a challenge, simply sign up for a free account on Frontend Mentor, browse through the available challenges, and select one that matches your skill level. Download the challenge files, which include design assets and a style guide, and start building the project on your local environment.
Can I use any technology or framework for the challenges?

How do I submit my solution to a challenge?
After completing a challenge, you can submit your solution by uploading your project to a platform like GitHub and sharing the live URL (if hosted on platforms like GitHub Pages, Netlify, or Vercel). Then, go to the challenge page on Frontend Mentor and submit the GitHub repository link and live URL. This allows others to view your solution and provide feedback.
